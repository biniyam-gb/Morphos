# Morphos

> Interactive mathematical visualization tool

![demo](assets/demo.gif)

---

## Options

Complex functions, Phase portraits, Fourier, Chaos, Fractals, Elliptic curves, Hyperbolic geometry, Number Theory, Linear algebra, Parametric surfaces, formal grammars, etc etc ...

## Running

Create a server

```bash
cd morphogen
python3 -m http.server 5500
# then open localhost:5500 in browser
```
